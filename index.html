各授業回の学習事項

第1回 WWW と HTML

今日は文字の書き方や、画像の出し方、見出しなどを習いました。 文字の書き方は<p>を使うとかけることを習いました。 また終了するときは</p>を使うことを習いました。 見出しは<h1>を使うと大きくなることを習いました。 そして数字を小さくしていくほど見出しが小さくなっていきます。 画像の出し方は、imgを使うとできることを習いました。

演習の成果: 演習1-4:のさまざまな要素

第2回 CSS による Web ページの装飾

今日はcssについて学びました。 cssとは、文字の色を変えられたり、背景の色、グラデーションなどの デザインを変えられたりします。 また、点線やドットなどもつけられたりします。 グリッドレイアウトでは、２×２（数字を変えれば違う数の表が作れる） の表も作れることを学びました。 そのおかげで、おしゃれなWebページを作れるようになりました。

演習の成果: ウェブブラウザの紹介

第3回 javaScript の基本

今日はJavaScriptについて学びました。 文字を表示させるときはconsoleを使うことを習いました。 また、コンソールやデバックを使い間違えを知れることも知りました。 ブレークポインについても学びました。 ブレークポイントは指定したとこで一時停止できることを知りました。

演習の成果: 演習3-6
